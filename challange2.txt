Create ProfileScreen.js, which will represent the profile of a user. The Screen should be built by two components, StudentInfo(full name, position, description, profile image) & Projects(image).

Profile Info - will be the first component being developed. It will include props as fullname, position, description, profile image. All of this data needs to be passed by component.

Projects - will be another component that will include only one prop, and that is an image that will be rendered and passed to screen
